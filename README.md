# Dealio

Dealio is a gaming-focused marketplace and esports event dashboard built with React, Vite, Tailwind CSS, Framer Motion, Lucide icons, and Three.js.

## Features

- Animated landing page with live tactical background effects
- Product armory with filtering, sorting, product details, and cart flow
- Login-gated tournament registration logic
- Real-time matches page with live countdowns, status filters, and search
- Responsive dark esports-inspired UI
